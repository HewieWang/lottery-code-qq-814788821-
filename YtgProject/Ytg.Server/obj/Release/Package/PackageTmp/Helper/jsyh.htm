• 1• 点击“账户充值”选项，选择“中国建设银行”，输入所要充值金额，获取平台工行收款人姓名 ，收款帐号；

    客户点击复制时,如果出现不允许访问或者错误时,很有可能是客户的浏览器设置问题（很有可能是火狐浏览器），我们建议客户使用其他浏览器再进行操作

  • 2• 登陆建设银行网站；https://ibsbjstar.ccb.com.cn/app/V5/CN/STY1/login.jsp

  • 3• 登陆后点击“转账汇款”,然后点击“活期转账汇款, 跳转跨行转账页面；

  • 4• 填写转账信息页面,填入在平台上获取的【建行】收款人姓名,收款人账号及金额,点击“下一步”；

  • 5• 跳转至确认转账信息页面，确认信息无误后点击“确认”按钮；

  • 6• 确认后需输入建行“网银盾密码”进行支付（为了您的资金安全，请用软键盘进行输入）；

  • 7• 支付信息确认后，进入个人中心的充值记录中查看游戏币到账情况。
